# notSoEasyTaxi


necesario
```
sudo apt-get install nodejs
```

verifico lo basico necesario
```
node -v
npm -v
```

despues de descargado el repo
```
npm install
```

ejecutar front
```
npm start
```
ejecutar back (para hacer cambios)
```
npm run dev
```
